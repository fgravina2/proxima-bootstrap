Sandboxed CSS

That's how it would work. Inside "sandbox" the user can upload their own CSS code, and it will be limited inside a personal ID. Example: "skin-1772".
The ID must be added before every user's rule. If the user puts "div a {}", it must be translated to "#skin-1772 div a {}". IDs must be blocked for input, because it would collapse against other used IDs throught the website.
All CSS properties must have "!important" added in each property. CSS classes can be used. This one simply changes the margin and background, the text centering was set by "textcenter" class.
Div with class "fixed", hey, it now has relative positioning!
